# Ad Performance Dashboard (Power BI)

## Objective
To analyze digital ad performance and identify opportunities to improve conversions and ROI.

## KPIs
- Impressions
- Click Through Rate (CTR)
- Conversions
- Conversion Drop-off

## Tools Used
- Power BI
- DAX
- Excel (data source)

## Key Insights
- High CTR of 11.76% indicates strong ad creatives
- Conversion drop observed post engagement
- Audience and geography analysis helped optimize targeting
